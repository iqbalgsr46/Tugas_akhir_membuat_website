# Tugas akhir membuat halaman website menggunakan teknik semantic HTML dan layouting

Deskripsi

Proyek ini adalah tugas akhir berjudul "Membuat Website". Tujuan proyek adalah merancang dan mengimplementasikan sebuah website responsif yang menampilkan informasi, fitur interaktif, dan desain yang rapi. Proyek ini dibuat untuk memenuhi persyaratan akademik dan sebagai portofolio pembelajaran.

Fitur

- Responsif di perangkat mobile dan desktop
- Penggunaan HTML, CSS

Teknologi

- HTML5
- CSS3
